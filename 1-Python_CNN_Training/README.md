# Lab 1

Lab 1 is the starting point for the ML side of the course work. It contains the initial notebook/PDF template, image data, logs, trained model data, and training outputs used to understand the model before moving inference toward C++ and the ZedBoard.

## What To Look At

- `tylerlab1/CprE_487_587_Lab1_Template.ipynb`: main notebook work.
- `tylerlab1/CprE_487_587_Lab1_Template.pdf`: exported notebook/report view.
- `tylerlab1/img_data/`: image inputs.
- `tylerlab1/model_data/`: model artifacts.
- `tylerlab1/train_data/`, `logs/`, `log_data/`: training and TensorBoard-style outputs.
- `lab1_08.zip`: submitted snapshot.

## Why This Lab Matters

This lab establishes the model-development workflow: prepare data, train or inspect the network, and save artifacts that later labs can compare against when implementing inference in C++ and hardware.
